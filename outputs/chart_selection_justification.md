# Chart Selection Justification

## 1. Sales Trend View

**Question Answered:** How are sales changing over time?

**Chart Type:** Line Chart

**Reason:** Line charts clearly show trends, seasonality, and growth patterns across time periods.

**Fields Used:**

* X-Axis: Order Date
* Y-Axis: Sales
* Filter: Region, Category, Segment

**Design Principle Applied:** Trend visualization and temporal analysis.

**Mistake Avoided:** Avoided pie charts which cannot effectively display time-based trends.


## 2. Regional Performance View

**Question Answered:** Which regions perform best?

**Chart Type:** Horizontal Bar Chart

**Reason:** Bar charts make regional comparisons easy and accurate.

**Fields Used:**

* Region
* Sales
* Profit
* Color by Profit

**Design Principle Applied:** Easy comparison of categorical values.

**Mistake Avoided:** Avoided 3D charts that distort comparisons.


## 3. Category Profitability View

**Question Answered:** Which categories and sub-categories generate profit?

**Chart Type:** Bar Chart

**Reason:** Supports comparison of profitability across multiple categories.

**Fields Used:**

* Category
* Sub-Category
* Profit
* Color by Profit

**Design Principle Applied:** Clear ranking and comparison.

**Mistake Avoided:** Avoided excessive color usage and clutter.



## 4. Discount vs Profit View

**Question Answered:** How do discounts affect profit?

**Chart Type:** Scatter Plot

**Reason:** Scatter plots effectively reveal relationships and correlations between variables.

**Fields Used:**

* Discount
* Profit
* Category

**Design Principle Applied:** Relationship analysis.

**Mistake Avoided:** Avoided aggregated charts that hide transaction-level patterns.


## 5. Return Analysis View

**Question Answered:** Which categories or segments have higher returns?

**Chart Type:** Bar Chart

**Reason:** Allows easy comparison of return rates.

**Fields Used:**

* Category
* Return Status
* Segment

**Design Principle Applied:** Clear categorical comparison.

**Mistake Avoided:** Avoided unnecessary visual complexity.


## Dashboard Design Principles

* Consistent color palette used across views.
* Interactive filters improve exploration.
* Clear titles and labels improve readability.
* KPI cards provide quick executive insights.
* Dashboard layout minimizes clutter.
* Proper sorting improves interpretation.
* Filter actions support interactive analysis.
* Business-focused visual hierarchy guides decision making.
